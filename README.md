## Run Locally

Clone the project

```bash
  git clone https://github.com/bhandaridhyeyh/ECHO.git
```

Go to the project directory

```bash
  cd Frontend
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run android
```
